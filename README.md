#  UpsMonitor — Программный индикатор уровня заряда ИБП

Приложение на стыке **Tkinter** и **Pygame** для имитации и мониторинга работы источника бесперебойного питания (ИБП) в режиме разряда аккумуляторной батареи. 

---

##  Возможности

* **Многовкладочный интерфейс (Tkinter):**
  * Главное графическое окно разделено на две вкладки («Управление» и «Инфо»).
  * Интерактивные элементы управления: кнопка «Работа от батареи» и ползунок (`Scale`) для точной настройки начальной емкости аккумулятора.
* **Объектно-ориентированная архитектура (ООП):**
  * Вся логика приложения инкапсулирована в рамках управляющего класса `UpsMonitor`.
* **Графическая визуализация (Pygame):**
  * Реалистичная отрисовка вертикального контура аккумуляторной батареи.
  * Анимация плавного ежесекундного уменьшения (опускания вниз) уровня оставшегося заряда.
* **Динамическая индикация состояний:**
  * **Выше 50%** — стабильный зеленый цвет индикатора.
  * **От 20% до 50%** — стабильный желтый цвет индикатора.
  * **Ниже 20%** — аварийный красный мигающий режим.
* **Система уведомлений:**
  * Мгновенный вывод стандартного всплывающего окна `messagebox` с предупреждением при достижении полного разряда (0%).

##  Установка зависимостей

### Вариант 1: Через менеджер пакетов MSYS2 UCRT64 (Рекомендуется)
Если вы запускаете проект в окружении MSYS2, установите готовую бинарную сборку Pygame:
```bash
pacman -S mingw-w64-ucrt-x86_64-python-pygame
```

### Вариант 2: Через стандартный менеджер пакетов Python (pip)
```bash
pip install pygame
```

---

##  Запуск приложения

Убедитесь, что файлы проекта (`sevenque.py` и `sevenque1.py` или один объединенный файл) находятся в одной рабочей директории, после чего выполните команду в консоли:

```bash
python sevenque1.py
```

---

##  Структура проекта

* `sevenque1.py` — отвечает за запуск GUI-интерфейса Tkinter, управление вкладками, обработку событий ползунка емкости и вызов графического окна.
* `sevenque.py` — содержит изолированный класс для обработки анимации разряда аккумулятора, логику циклов Pygame, таймеры и динамическую смену цветов.
